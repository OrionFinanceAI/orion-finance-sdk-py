# Orion Finance SDK

Welcome to the documentation for the Orion Finance Python SDK.

## Installation

```bash
pip install orion-finance-sdk-py
```

## Quick Start

```python
from orion_finance_sdk_py import OrionConfig

config = OrionConfig()
print(f"Risk-free Rate: {config.risk_free_rate}")
```
